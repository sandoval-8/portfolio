Gestor de Asistencias

Aplicación web encargada de llevar el registro de las asistencias mensuales . Es un proyecto Spring boot, desarrollado con el lenguaje java en su version 11, utiliza la seguridad de Spring Security con Json Web Token (JWT), los empleados son registrados en una base de datos MySQL los cuales son obtenidos mediante una Api REST y autocompletado del formulario mediante AJAX.
